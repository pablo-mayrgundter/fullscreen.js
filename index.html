fullscreen.js

Fullscreen is a small (2KB) utility to resize/toggle DOM elements to full-screen.

Click the like this at the bottom right.

Browser Compatibility

  • Chrome 88/Brave 1.19 ✓
  • Safari 13.1 ✓
  • Firefox 84 ✓

Patches to normalize cross browser appreciated! See https://github.com/pablo-mayrgundter/fullscreen.js
